# Research Paper Workspace

This folder contains the draft material, publication-quality figures, compiled tables, equation lists, and reference files for preparing the research manuscript.

## Directory Structure

```
paper/
├── IEEE_Manuscript.md         # Manuscript outline skeleton (IEEE style sections)
├── README.md                  # This description file
├── equations/
│   └── Equations_Summary.md   # Mathematical equations listing and code mapping
├── figures/
│   ├── Figure_Index.md        # Figure catalog with description and sources
│   ├── simulation/            # Figures 1 to 7 (Baselines & Physics studies)
│   ├── machine_learning/      # Figures 8 to 12 (Dataset, Performance, SHAP)
│   └── optimization/          # Figure 13 (Differential Evolution search)
├── tables/
│   ├── Table_Index.md         # Table index mapping CSV contents
│   ├── Table_1_Simulation_Parameters.csv
│   ├── Table_2_Model_Hyperparameters.csv
│   ├── Table_3_Model_Evaluation_Metrics.csv
│   ├── Table_4_Cross_Validation_Scores.csv
│   └── Table_5_Optimization_Results.csv
└── references/
    ├── citation_checklist.md  # Citation targets tracker
    ├── reference_notes.md     # Literature review notes
    └── references.bib         # BibTeX literature database
```
