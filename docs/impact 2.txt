PLGA NANOPARTICLE SIMULATION
PARAMETER IMPACT GUIDE

====================================================

1. DOMAIN PARAMETERS
   ====================================================

Lx = Tumor Length
Ly = Tumor Width

Effect of Increasing:

* Larger simulation domain
* Longer nanoparticle travel distance
* Increased computational cost

Effect of Decreasing:

* Smaller tumor domain
* Faster simulation

---

Nx = Number of Grid Points (X)
Ny = Number of Grid Points (Y)

Effect of Increasing:

* Higher numerical accuracy
* Smoother contour plots
* Increased runtime

Effect of Decreasing:

* Faster computation
* Reduced accuracy

====================================================
2. NANOPARTICLE TRANSPORT PARAMETERS
====================================

D = Diffusion Coefficient

Current Value:
D = 1e-8

Increasing D:

* Nanoparticles spread faster
* Greater penetration depth
* Wider concentration distribution

Decreasing D:

* Slower diffusion
* Reduced penetration depth

Primary Impact:
Result 1 (Particle Size Study)

---

K = Hydraulic Conductivity

Current Value:
K = 1e-13

Increasing K:

* Stronger fluid flow
* Increased convection
* Faster transport

Decreasing K:

* Diffusion dominates transport
* Reduced fluid movement

Primary Impact:
Pressure and Velocity Fields

---

uptake_rate = Cellular Uptake Rate

Current Value:
0.02

Increasing:

* Nanoparticles are absorbed quickly
* Reduced penetration depth
* Less drug reaches deeper tissue

Decreasing:

* Nanoparticles travel further
* Increased penetration depth

Primary Impact:
Result 1
